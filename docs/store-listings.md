# Store Listing материал — DIY Store аппууд

App Store болон Google Play-д submit хийхэд хэрэгтэй бүх текст. Эхэлж гаргах: **Customer + Supplier**. Driver дараа нь.

Тэмдэглэл талбарын урт:
- **App Store:** subtitle ≤30 тэмдэгт, keywords ≤100 тэмдэгт (таслалаар), promotional text ≤170, description ≤4000.
- **Google Play:** short description ≤80 тэмдэгт, full description ≤4000.
- Холбоо: support@shoptool.mn · Privacy: https://shoptool.mn/privacy · Website: https://shoptool.mn

---

## Content Rating (хоёр store)

Бүх апп нь барилгын материалын худалдаа — насанд хүрээгүйд тохиромжтой, хүчирхийлэл/насанд хүрэгчдийн контентгүй.

**Google Play (IARC асуулга) — хариултууд:**
- Хүчирхийлэл, цус: **Үгүй**
- Бэлгийн контент: **Үгүй**
- Хараал, бүдүүлэг үг: **Үгүй**
- Хар тамхи/архи/тамхи: **Үгүй**
- Мөрийтэй тоглоом: **Үгүй**
- Хэрэглэгч хоорондын харилцаа (chat): хүргэлтийн чат байгаа бол **Тийм** (зөвхөн захиалгын хүрээнд)
- Байршил хуваалцах: **Тийм** (хүргэлтэд)
- → Үр дүн: ихэвчлэн **Everyone / 3+**

**App Store age rating:** 4+ (онцгой контентгүй)

## Data Safety / Privacy (хоёр store)

Цуглуулдаг өгөгдөл (privacy page-тэй нийцүүлэх — https://shoptool.mn/privacy):

| Өгөгдөл | Зорилго | Аппууд |
|---------|---------|--------|
| Нэр, и-мэйл, утас | Бүртгэл, нэвтрэлт (OTP) | Бүгд |
| Байршил | Хүргэлт, явц хянах | Customer, Driver |
| Зураг/камер | Бараа байршуулах | Supplier |
| Хүргэлтийн хаяг | Захиалга гүйцэтгэх | Customer |
| Төлбөрийн мэдээлэл | QPay/MonPay/карт (бид карт хадгалдаггүй) | Customer |

- Өгөгдлийг **зардаггүй** (not sold)
- Шифрлэлттэй дамжуулна (HTTPS)
- Хэрэглэгч устгуулах хүсэлт гаргах боломжтой → support@shoptool.mn

## Screenshot шаардлага (#3)

Screenshot-уудыг preview/production build суулгасан утас эсвэл simulator-аас авна. Доорх дэлгэцүүдийг санал болгож байна:

### Google Play (заавал)
- **Утасны screenshot:** хамгийн багадаа 2, дээд тал 8. Хэмжээ: 16:9 эсвэл 9:16, талын урт 320–3840px.
- **Feature graphic:** 1024×500 PNG/JPG (заавал). Лого + товч уриа.
- **App icon:** 512×512 (32-bit PNG).

### App Store (заавал)
- **6.7" iPhone** (1290×2796) — заавал, дор хаяж 3
- **6.5" iPhone** (1242×2688) — сонголтоор
- (App Store одоо 6.7" болон 6.5"-г л шаардана; 5.5" заавал биш)

### Аль дэлгэцийг авах вэ
**Customer:** Нүүр (ангилал/бараа) · Хайлт · Барааны дэлгэрэнгүй · Сагс · Захиалга хянах (газрын зураг)
**Supplier:** Самбар (dashboard) · Барааны жагсаалт · Бараа нэмэх (зурагтай) · Захиалга · Орлого/тайлан
